# 仙暉 HR 月結考勤檢核系統

這是一個可直接用瀏覽器開啟的 HR 考勤月結檢核網頁工具。

## 主要功能

- 匯入 Excel / CSV 考勤資料
- 依月份與人員整理月結考勤
- 自動補齊 1 號到月底每日資料
- 標示未上班 / 休假日
- 每日未滿 9 小時以紅字提醒
- 月加班超過 46 小時後以紅字提醒
- 匯出帶有顏色提醒的 Excel 檢核表

## 使用方式

1. 用瀏覽器開啟 `index.html`
2. 點選 `匯入 Excel`
3. 選擇考勤資料檔案
4. 檢查月份、人員與每日明細
5. 點選 `匯出檢核 Excel`

## Excel 欄位

建議 Excel 第一列包含以下欄位：

| 欄位 | 說明 |
|---|---|
| 員工序號 | 員工識別 |
| 考勤號碼 | 考勤系統編號 |
| 姓名 | 員工姓名 |
| 日期 | 考勤日期 |
| 上班1 | 第一段上班時間 |
| 下班1 | 第一段下班時間 |
| 上班2 | 第二段上班時間 |
| 下班2 | 第二段下班時間 |
| 合計 | 每日總工時 |
| 加班時數 | 每日加班時數 |
